Gráfico de relacionamento entre deputados(as) e CNPJ

Após clicar em atualizar, aguarde ~15 segundos pro gráfico ser gerado


Perguntas frequentes

1. Qual é a fonte dos dados?

Arquivos disponibilizados pelo site Dados Abertos mantido pelo governo.

2. Como são definidas as linhas?E as entidades?

Cada linha do gráfico representa um relacionamento entre um CPF e um CNPJ dos arquivos .CSV da categoria "Despesas pela Cota para Exercício da Atividade Parlamentar", disponibilizado no Dados Abertos. A entidade central corresponde ao CPF e as adjacentes aos CNPJs.

3. O valor (R$) da despesa é levado em conta na construção do gráfico?

Não, as linhas se referem a quantidade de despesas CPF vs CNPJ e não ao valor das despesas.

4. Por que não posso selecionar as despesas de todos os anos para um candidato?

Juntar todos os anos em um só gráfico causa lentidão no carregamento do gráfico, por isso preferi separar por ano.

5. No site Dados Abertos tem despesas desde 2008, porque só consigo filtrar a partir de 2019?

Quando tiver um tempo vou atualizar a interface, mas você pode alterar o endereço (URL) pelo CPF e ano em que o candidato esteve eleito, por exemplo, Jair Bolsonaro era deputado federal em 2017, altere a URL para /index.html?&dep=45317828791&ano=2017

7. Por que algumas entidades estão sem relacionmento(soltas no gráfico)?

Por algum motivo, algumas empresas no arquivo fonte estão com o CNPJ vazio, por isso elas estão soltas no gráfico. Acesse o arquivo .csv no LibreOffice/Excel para verificar mais detalhes sobre essas entidates.

8. Por que existem entidades com nomes parecidos?

Porque é assim que está no arquivo .csv 🤣

6. Como faço pra explorar mais os dados do site Dados Abertos?

Para acessar os dados brutos utilizados para gerar os gráficos, é só baixar os arquivos .CSV referenciado na pergunta 2 e abrir no LibreOffice/Excel. No arquivo é possível verificar valores das notas, datas, link para o arquivo da nota fiscal completa, etc. Para analisar outros dados, pode começar por esta outra página www.camara.leg.br/transparencia/gastos-parlamentares
